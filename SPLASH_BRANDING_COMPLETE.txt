🐛 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

ISSUE: Splash doesn't appear
  SOLUTION:
  1. Check framer-motion is installed: npm ls framer-motion
  2. Check SplashProvider in layout.tsx
  3. Clear browser cache
  4. Check console for errors
  5. Try incognito mode (clears sessionStorage)

ISSUE: Animations are janky
  SOLUTION:
  1. Enable GPU acceleration in DevTools
  2. Test in different browser
  3. Check DevTools Performance tab
  4. Reduce animation duration for testing
  5. Check CPU/GPU usage

ISSUE: Splash appears every time
  SOLUTION:
  1. Clear sessionStorage in DevTools
  2. Check sessionStorage isn't being cleared
  3. Verify splash-seen flag is being set
  4. Test on different device

For more help:
  → See SPLASH_SCREEN_GUIDE.md (Troubleshooting section)
  → Review component code comments
  → Check Framer Motion documentation
