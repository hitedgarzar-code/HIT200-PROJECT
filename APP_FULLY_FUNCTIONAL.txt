========================================================================================
                              HOW TO GET STARTED
========================================================================================

1. SETUP ENVIRONMENT
   - Copy .env.example to .env.local
   - Add your Supabase URL and keys
   - Add NEXT_PUBLIC_APP_URL=http://localhost:3000

2. INSTALL & RUN
   $ pnpm install
   $ pnpm dev

3. VISIT IN BROWSER
   http://localhost:3000

4. TEST THE FLOW
   - View home page (featured products)
   - Browse /shop (all 17 products)
   - Click product for details
   - Signup/Login
   - Add to cart
   - Checkout with payment
