3. SAVE RECEIPTS WITH AUTO-INCREMENT NUMBERS ✓
================================================================================

RECEIPT NUMBER GENERATION:
- Starts from 1 (REC-000001, REC-000002, etc.)
- Atomically incremented in database function
- Format: REC-XXXXXX (6-digit padded numbers)
- Auto-generated when order is completed

DATABASE SETUP:
- receipt_sequence table tracks last number
- PostgreSQL function get_next_receipt_number() increments atomically
- API route /api/receipts/generate generates new numbers
- No duplicate numbers possible

RECEIPT FEATURES:
- Automatic generation after successful payment
- Unique receipt number for each order
- Includes:
  * edgARs logo and branding
  * Receipt number and order date
  * Customer information
  * All items purchased with sizes and quantities
  * Subtotal, shipping, tax, total
  * Payment method and reference
  * Professional formatted layout
