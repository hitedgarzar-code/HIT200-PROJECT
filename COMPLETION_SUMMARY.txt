QUICK START INSTRUCTIONS
================================================================================

1. INSTALL DEPENDENCIES
   $ pnpm install

2. CREATE ENVIRONMENT FILE
   $ cp .env.example .env.local
   Add your Supabase credentials

3. SETUP DATABASE
   - Go to Supabase Dashboard
   - SQL Editor > New Query
   - Paste scripts/001_create_tables.sql
   - Execute

4. SEED PRODUCTS
   $ node scripts/seed-products.js

5. RUN DEVELOPMENT SERVER
   $ pnpm dev
   Visit: http://localhost:3000

6. TEST THE APP
   - Sign up: http://localhost:3000/auth/signup
   - Verify email (check Supabase auth)
   - Browse products: http://localhost:3000/shop
   - Add to cart and checkout
   - View orders: http://localhost:3000/orders
   - Admin: Make yourself admin, visit /admin

7. DEPLOY (See DEPLOYMENT.md)
   - Push to GitHub
   - Connect to Vercel
   - Set environment variables
   - Deploy!
