# edgARs Branding Guide

## Brand Overview

**Brand Name:** edgARs (stylized with emphasis on "AR")  
**Industry:** Premium Fashion Retail  
**Market:** Zimbabwe & Southern Africa  
**Tagline:** "Premium Fashion Meets Innovation"

## Visual Identity

### Logo

The edgARs logo features:
- **Letter "e"** in a gold (#d4a574) circle badge
- **Brand name** displayed as "edg**AR**s" with "AR" highlighted in gold
- Modern, minimalist design reflecting cutting-edge technology and fashion

**Logo Usage:**
```
Standard: edgARs (with AR in accent color)
Short: e (in gold circle)
Full: [Logo circle] edgARs
```

### Color Palette

#### Primary Colors
- **Primary Navy:** #1a2b4d (Hex) | oklch(0.2 0.04 240)
  - Used for: Headers, navigation, primary buttons
  - Represents: Sophistication, trust, premium quality

- **Accent Gold:** #d4a574 (Hex) | oklch(0.65 0.12 50)
  - Used for: Highlights, "AR" in logo, accent elements
  - Represents: Luxury, elegance, innovation

#### Secondary Colors
- **Primary Light:** #2d3e5f - Lighter navy for hover states
- **Primary Dark:** #0f1923 - Darker navy for depth
- **Accent Light:** #e8c4a0 - Lighter gold for subtle accents
- **Accent Dark:** #a67c52 - Darker gold for emphasis

#### Neutral Palette
- **Background:** #fafafa (Off-white)
- **Foreground:** #1a1a1a (Near black)
- **Neutral 100:** #f5f5f5
- **Neutral 200:** #eeeeee
- **Neutral 300:** #e0e0e0
- **Neutral 400-800:** Grays for various UI elements

### Typography

#### Font Family
- **Sans-serif (Primary):** Geist
  - Used for: Body text, UI elements, most content
  - Weight: 400 (Regular), 600 (Semibold), 700 (Bold)

- **Monospace (Secondary):** Geist Mono
  - Used for: Code, technical content, pricing displays

#### Type Scale
- **H1 (Hero):** 56px (Desktop) / 40px (Mobile) - Bold
- **H2 (Section):** 48px - Bold
- **H3 (Subsection):** 32px - Semibold
- **H4 (Subheading):** 24px - Semibold
- **Body:** 16px - Regular (line-height: 1.6)
- **Caption:** 12px - Regular
- **Label:** 14px - Medium

#### Text Colors
- **Primary Text:** #1a1a1a (foreground)
- **Secondary Text:** #555555 (muted-foreground)
- **Links:** #d4a574 (accent color)
- **On Dark:** #ffffff (white)

### Spacing System

Using Tailwind CSS spacing scale:
- **xs:** 4px (1 unit)
- **sm:** 8px (2 units)
- **md:** 16px (4 units)
- **lg:** 24px (6 units)
- **xl:** 32px (8 units)
- **2xl:** 48px (12 units)
- **3xl:** 64px (16 units)

### Border Radius

- **Default:** 6px (0.375rem)
- **Rounded:** 8px
- **Full:** 9999px (for circles)

## Writing Style

### Brand Voice
- **Tone:** Premium, aspirational, contemporary
- **Personality:** Sophisticated yet approachable
- **Language:** Clear, concise, modern

### Key Messaging
- "Premium Fashion Meets Innovation"
- "Redefining fashion with style and elegance"
- "Where technology and fashion converge"
- "Elevate your wardrobe"

### Grammar & Usage
- Use sentence case for headings
- Use active voice
- Be specific and avoid jargon
- Keep CTAs action-oriented

## Imagery & Photography

### Product Photography
- **Style:** Clean, well-lit, professional
- **Background:** White or neutral tones
- **Consistency:** Uniform lighting and angles across catalog
- **Quality:** High-resolution (minimum 1000px width)

### Color Treatment
- **Saturation:** Natural, not oversaturated
- **White Balance:** Neutral white tones
- **Editing:** Minimal, maintain product authenticity

## Do's and Don'ts

### Do's ✓
- Use navy and gold consistently
- Maintain clear spacing around logo
- Use professional, high-quality imagery
- Apply consistent typography hierarchy
- Ensure color contrast for accessibility
- Use primary colors for interactive elements
- Keep designs clean and minimal

### Don'ts ✗
- Don't change the brand name spelling
- Don't use other colors as primary
- Don't stretch or distort logo
- Don't use low-quality or blurry images
- Don't mix multiple font families
- Don't apply poor color contrast
- Don't over-decorate or clutter
- Don't use outdated imagery styles

## Accessibility

### Color Contrast
- **AAA Level** (4.5:1 minimum):
  - Navy (#1a2b4d) on White (#ffffff) ✓
  - White (#ffffff) on Navy (#1a2b4d) ✓
  - Gold (#d4a574) on White (#ffffff) ✓

### Typography Accessibility
- Minimum font size: 14px for body text
- Line-height: 1.6 for readability
- Avoid all-caps for large text blocks
- Use semantic HTML headings

### Interactive Elements
- Touch targets: Minimum 48px × 48px
- Button labels: Clear and descriptive
- Focus states: Visible (gold outline)
- ARIA labels: For screen readers
