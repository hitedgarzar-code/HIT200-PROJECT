QUICK START
================================================================================

1. Install dependencies:
   $ pnpm install

2. Setup environment (.env.local):
   NEXT_PUBLIC_SUPABASE_URL=your_url
   NEXT_PUBLIC_SUPABASE_ANON_KEY=your_key
   SUPABASE_SERVICE_ROLE_KEY=your_key

3. Create admin and sample data:
   $ node scripts/setup-admin.js

4. Start development server:
   $ pnpm dev

5. Open browser:
   http://localhost:3000

6. Login to admin dashboard:
   Email: admin@edgars.com
   Password: AdminEdgars2024!

PAYMENT METHODS
================================================================================

PayNow (Instant Payment):
- Processing time: 30 seconds to 2 minutes
- Supported wallets: EcoCash, OneMoney, InnBucks
- Status: Completed immediately
- Fees: As per PayNow provider
- Implementation: Real-time polling

Pay Later (Flexible Instalments):
- Plan options:
  * 3 months (5% interest)
  * 6 months (8% interest)
  * 12 months (12% interest)
- First payment: 30 days from order
- Automatic reminders: Monthly
- Late fees: 2% per month
- Full implementation: Complete with UI and tracking
