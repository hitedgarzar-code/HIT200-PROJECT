SUPPORT & TROUBLESHOOTING
================================================================================

ISSUE: Splash screen doesn't appear
SOLUTION:
1. Check: framer-motion is installed
2. Check: SplashProvider is in layout.tsx
3. Clear: Browser cache
4. Verify: No console errors
5. Test: Incognito mode (sessionStorage clear)

ISSUE: Animations are janky
SOLUTION:
1. Check: Hardware acceleration enabled
2. Test: Different browser
3. Reduce: Animation duration for testing
4. Monitor: DevTools Performance tab
5. Check: CPU/GPU usage

ISSUE: Splash appears every time
SOLUTION:
1. Clear: sessionStorage in DevTools
2. Check: sessionStorage not being cleared
3. Verify: Script is saving splash-seen
4. Test: Different device/browser

ISSUE: Framer Motion not found
SOLUTION:
1. Run: pnpm install
2. Check: pnpm ls framer-motion
3. Try: Delete node_modules and reinstall
4. Verify: package.json has dependency

For more help:
- See: SPLASH_SCREEN_GUIDE.md
